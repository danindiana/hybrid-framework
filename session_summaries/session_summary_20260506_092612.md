## 1. Objective
Remove all legacy configurations and scripts related to the Intel X540-T2 NIC (`enp3s0f0`, `enp3s0f1`) which has been physically removed from the system.

## 2. Cleanup Actions

### 2.1 System Configurations
- **Udev Rules:** Deleted `/etc/udev/rules.d/99-x540-optimize.rules` which was attempting to set ring buffers and coalescing on the missing PCI devices.

### 2.2 Workspace Scripts
- **`repair_network.sh`:** Updated the routing cleanup logic to dynamically detect the active interface instead of targeting `enp3s0f0/1`. It now safely clears any rogue multipath routes on the current UP interface.
- **`optimize_network_queues.sh`:** (Created earlier this session) Already uses dynamic detection and is compatible with the current hardware (`enp7s0`).
